# File CRUD Operations in File Explorer

**Date**: 2026-04-04
**Status**: Complete

## What was built

Right-click context menu on file tree entries with full CRUD operations.

### Context Menu
- Dark themed popup (position: fixed, z-index 1000) with shadow
- Appears on right-click of any file or folder
- Dismisses on click outside

### Operations
- **New Note**: Creates empty .md file, inline input for naming, auto-opens in tab
- **New Folder**: Creates folder via placeholder file, inline input for naming
- **Rename**: Inline input replacing the filename, pre-selects name without extension
- **Delete**: Confirmation dialog, then DELETE request (red text for danger)

### Inline Input
- Purple border matching app accent
- Auto-focused with filename selected (excluding extension)
- Submit on Enter, cancel on Escape, submit on blur
- Positioned inline in the tree at correct depth

### Tree Refresh
- `refreshTree` callback passed as `onMutate` prop
- Automatically re-fetches tree from server after any mutation
- New files are auto-opened in a tab after creation
