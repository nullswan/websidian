# Auto-Save in Editor

**Date**: 2026-04-04
**Status**: Complete

## What was built
- Editor auto-saves content 1.5 seconds after the last keystroke
- Works alongside existing Ctrl+S manual save
- Debounced timer resets on each change to avoid excessive saves
- Timer cleaned up on component unmount

## Implementation
- Added `EditorView.updateListener` extension in Editor.tsx
- On `docChanged`, clears previous timer and sets new 1.5s timeout
- Calls existing `onSave` callback (same as Ctrl+S)
- Uses a ref to track the timer across renders

## Files changed
- `packages/web/src/components/Editor.tsx` — auto-save update listener, timer ref, cleanup
