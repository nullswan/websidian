# Zen Mode (Distraction-Free Writing)

**Date**: 2026-04-05
**Status**: Complete

## What was built

### Distraction-free writing toggle
- `Ctrl+Shift+Z` toggles zen mode on/off
- Hides: ribbon, left sidebar, right sidebar, tab bar, status bar
- Breadcrumb remains visible for orientation
- `Escape` exits zen mode
- Sidebar collapsed states saved before entering, restored on exit
- Available via command palette ("Enter Zen Mode" / "Exit Zen Mode")
- Listed in keyboard shortcuts overlay

## Key technical decisions
- **Pre-zen state ref**: `zenPrevState` ref stores `{ left, right }` collapsed booleans so exiting restores the exact prior sidebar layout
- **`display: "none"` on ribbon**: Cleaner than conditional rendering — avoids re-mounting the ribbon's buttons/icons on toggle
- **Guard conditions**: Tab bar uses `!zenMode && pane.tabIds.length > 0`, status bar uses `!zenMode && activeTab`
- **Shortcut choice**: `Ctrl+Shift+Z` avoids conflict with existing shortcuts (Ctrl+Shift+F is search, Ctrl+Z is undo)
