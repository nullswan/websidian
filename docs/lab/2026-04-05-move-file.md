# Move File to Folder

**Date**: 2026-04-05
**Status**: Complete

## What was built

### Move current file to another folder
- Command palette: "Move current file to..."
- Shows folder picker modal with search/filter
- Keyboard navigation (ArrowUp/Down, Enter, Escape)
- Current folder highlighted with accent color + "current" badge
- "(root)" option to move to vault root
- Reuses rename API — auto-updates wikilinks in other files

## Key technical decisions
- **FolderPicker as local component**: Defined in App.tsx alongside ScrollContainer/SidebarSection
- **IIFE in JSX**: Lazily computes folder list only when picker is open
- **Rename API reuse**: Move = rename with different path prefix, same endpoint handles both
